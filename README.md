# GGamma

GTK gamma adjustment GUI powered by [xrandr](https://www.x.org/releases/X11R7.5/doc/man/man1/xrandr.1.html)

Requires: xrandr, Python, & GTK3
Install: `python -m pip install git+git://github.com/thann/ggamma --user`
Run: `ggamma`
